# 轻量级日历

一个简洁优雅的桌面日历应用，支持备忘录和计划模式，帮助你高效管理时间和任务。

![版本](https://img.shields.io/badge/版本-1.0.5-blue)
![平台](https://img.shields.io/badge/平台-Windows-lightgrey)
![Electron](https://img.shields.io/badge/Electron-28.3.3-47848F?logo=electron)

## ✨ 功能特性

### 📅 日历视图
- **周视图显示**：以周为单位展示日历，清晰直观
- **自动同步**：与系统时间实时同步
- **快速导航**：轻松切换不同周次

### 📝 备忘录
- **日记模式**：为每一天记录备忘录
- **周记模式**：为每一周记录总结和计划
- **可视化标记**：有备忘录的日期会自动显示标记

### ✅ 计划模式
- **当日计划**：创建每日任务清单
- **完成追踪**：点击圆圈标记任务完成状态
- **未完成清单**：未完成的任务自动移入共享清单
- **拖拽管理**：将任务在计划和未完成清单间拖拽
- **双击编辑**：双击即可修改任务内容
- **周计划**：独立的周计划系统，与日计划隔离

### 🎨 界面设计
- **奶油色配色**：温暖的奶油黄、奶油橙色调
- **简洁 UI**：直观易用的界面设计
- **响应式布局**：自适应不同窗口大小

## 🚀 安装和使用

### 方法一：安装程序（推荐）

1. 下载 `轻量级日历 Setup 1.0.0.exe`
2. 双击运行安装程序
3. 按照安装向导完成安装
4. 从开始菜单或桌面快捷方式启动应用

### 方法二：开发模式运行

```bash
# 1. 安装依赖
npm install

# 2. 启动应用
npm start
```

### 方法三：自行打包

```bash
# 安装依赖
npm install

# 打包 Windows 版本
npm run build:win

# 安装包位置：dist/轻量级日历 Setup 1.0.0.exe
```

## 💾 数据存储

- **存储位置**：`C:\Users\你的用户名\AppData\Roaming\轻量级日历\calendar_data.json`
- **存储格式**：JSON
- **自动保存**：所有修改自动保存到本地文件
- **数据安全**：数据仅保存在本地，不会上传

## 🎯 使用说明

### 添加备忘录
1. 点击任意日期或周
2. 在弹出的对话框中输入内容
3. 点击"保存"按钮

### 使用计划模式
1. 点击"切换到计划模式"按钮
2. 添加任务到当日/本周计划
3. 点击任务后的圆圈标记完成状态
4. 未完成的任务可拖拽到下方清单
5. 双击任务可编辑内容

### 快捷键
- 当前无快捷键，后续版本可能添加
